# 풀서비스 출시 검수 자동화 도구

## 1. 프로젝트 개요

메가스터디 연간 시험 회차(3월·5월 학력평가, 6월·9월 모의평가, 7월·10월 학력평가, 11월 수능) 별로 풀서비스 출시 전후 AI검색 답변을 자동 수집·기록하는 내부 운영 도구입니다.

사용자가 설정한 시험 회차, 타임라인 단계, 테스트 질의 세트를 기준으로 실제 메가스터디 AI검색 페이지에 질의를 자동 입력하고, AI모드 답변 영역(`#ai-summary-container`)의 내용을 수집·기록합니다.

## 2. 참고 자료 확인 기준

작업 시작 시 `docs/` 폴더 내 파일 목록을 확인합니다.  
현재 확인된 파일:
- `3월 학력평가 풀서비스 시나리오_v1.9_김소연_260312.pdf`
- `핸즈온멘토링 개인 과제_최지석_260507.docx`

## 3. 1차 구현 범위

- 시험 회차 선택·추가·수정·삭제
- 시험별 타임라인 관리 (기본값 자동 로드, 수정, 저장)
- 테스트 질의 세트 관리 (엑셀 업로드, 직접 추가/수정/삭제, 다중 필터)
- 실제 메가스터디 AI검색 페이지 자동 검색 (Playwright)
- AI모드 `#ai-summary-container` 답변 수집
- 수집 결과 저장 (`data/saved/`)
- 수집 결과 테이블 조회 및 필터
- 결과 엑셀 다운로드 (4개 시트)

## 4. 구현하지 않은 범위

- 기준 데이터 입력 / 템플릿 다운로드
- 정답·배점·등급컷 데이터 관리
- 답변 정오 판단
- 적절/부적절/확인 필요 판정
- 리스크 등급 분류
- 수정 요청 문구 생성
- 재검수 대상 자동 분류
- AI검색 API 연동
- Streamlit 사용

## 5. 실행 방법

```bash
# 의존성 설치
npm install

# Playwright Chromium 브라우저 설치
npx playwright install chromium

# 개발 서버 실행
npm run dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000) 접속

환경 변수 설정 (선택):
```bash
# .env.example → .env.local 로 복사 후 수정
cp .env.example .env.local
```

## 6. 시험 회차 관리 방법

1. 화면 상단 **1. 시험 회차 선택** 섹션에서 라디오 버튼으로 회차 선택
2. **+ 회차 추가** 버튼 → 시험명, 유형, 연도, 월 입력 후 저장
3. **수정** 버튼 → 해당 회차 정보 수정
4. **삭제** 버튼 → 해당 회차 삭제

## 7. 시험별 타임라인 수정 방법

1. 시험 회차 선택 시 **2. 타임라인 관리** 섹션에 해당 회차 타임라인 자동 로드
2. 저장된 타임라인이 없으면 기본 15개 단계 자동 적용
3. 각 행에서 단계명, 예상 시각, 실제 반영 시각, 비고 수정 가능
4. ▲▼ 버튼으로 순서 변경
5. **실행 단계** 라디오 버튼으로 현재 검수 시점 선택
6. **저장** 버튼 클릭으로 `data/saved/exam_timelines.json`에 저장
7. **기본값 초기화** 버튼으로 기본 타임라인으로 되돌리기

## 8. 테스트 질의 세트 업로드 방법

### 엑셀/CSV 업로드

**필수 컬럼:**

| 컬럼명 | 설명 |
|---|---|
| query_id | 질의 고유 ID |
| category | 질의 유형 (정답/배점/등급컷/... 중 하나) |
| query_text | 실제 검색창에 입력할 질의 내용 |
| target_stage | 대상 타임라인 단계명 |
| priority | 우선순위 (high/medium/low) |
| poc_yn | POC 여부 (Y/N 또는 true/false) |

**선택 컬럼:** `exam_id`, `sub_category`, `note`

**category 허용값:**
정답, 배점, 등급컷, 정오답률, 난이도, 선지별 선택비율, 해설강의, 총평, 라이브 설명회, 다시보기, 풀서비스 경로, 예외/방어

**업로드 순서:**
1. **엑셀/CSV 업로드** 버튼 클릭
2. `.xlsx` 또는 `.csv` 파일 선택
3. 미리보기 확인 후 **저장** 클릭

### 직접 추가

**+ 직접 추가** 버튼 → 폼에서 각 필드 입력 후 저장

## 9. 실제 AI검색 답변 수집 방법

1. **2. 타임라인 관리**에서 현재 실행 단계 라디오 버튼 선택
2. **3. 테스트 질의 세트 관리**에서 실행할 질의 체크박스 선택 (또는 전체 선택)
3. **4. 검색 실행** 섹션에서 실행 모드 선택:
   - **선택한 질의만 실행**: 체크된 질의만
   - **전체 질의 실행**: 현재 시험 회차의 모든 질의
4. **검색 실행** 버튼 클릭
5. Playwright가 실제 브라우저를 열어 메가스터디 AI검색 페이지에 자동 접속·검색
6. 완료 후 **5. 수집 결과** 섹션에 결과 자동 표시

**수집 동작:**
- 검색창(`textarea#kwd2`)에 질의 입력 → Enter
- AI모드 탭 자동 클릭 (필요 시)
- `#ai-summary-container` 영역 최대 30초 대기
- 답변 제목·본문·링크 수집
- 실패 시 1회 재시도, 이후 실패로 기록하고 다음 질의 진행
- 질의 간 5~10초 대기

## 10. 결과 엑셀 다운로드 방법

**5. 수집 결과** 섹션 상단 **엑셀 다운로드** 버튼 클릭

파일명: `fullservice_ai_search_results_YYYYMMDD_HHMM.xlsx`

4개 시트:
1. `수집_결과` — 전체 수집 결과
2. `실행_로그` — 질의별 실행 로그
3. `테스트_질의세트` — 등록된 질의 목록
4. `시험별_타임라인` — 저장된 타임라인
